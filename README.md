# ГУФ x НОЙЗ — Pixel Fighter

Готовая браузерная версия для GitHub Pages.

## Как запустить локально
Открой `index.html` в браузере.

## Как залить на GitHub Pages
1. Создай репозиторий.
2. Загрузи все файлы и папку `assets`.
3. Settings → Pages → Deploy from branch → `main` / root.
4. Открой ссылку Pages.

## Управление
- A / D — ходьба
- W — прыжок
- J — слабый удар
- K — сильный удар
- L — спецприём
- На телефоне есть экранные кнопки.

## Что внутри
- `index.html`, `style.css`, `game.js` — игра.
- `assets/sprites` — нарезанные спрайты.
- `assets/backgrounds` — Москва и Таиланд.
- `assets/effects` — дым, звуковая волна, вспышка, пыль.
- `assets/ui` — заставка и UI.
- `assets/portraits` — портреты для выбора персонажа.
- `assets/source` — исходные сгенерированные листы и логотип.

## Важно
Спрайты нарезаны автоматически из sprite sheet, фон очищен алгоритмически. Для финальной версии лучше вручную проверить и почистить отдельные кадры.
