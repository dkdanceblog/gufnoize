v22 mobile UI fix

Что исправлено:
- полностью переделаны touch-кнопки для мобильного;
- кнопки больше не стоят одной длинной линией и не налезают друг на друга;
- слева теперь нормальный d-pad:
  jump / left / crouch / right
- справа треугольник атак:
  J / K / L
- canvas на мобильном автоматически уменьшается по высоте,
  чтобы оставить место под кнопки;
- добавлены safe-area отступы для телефонов с вырезами/жестовой навигацией.
